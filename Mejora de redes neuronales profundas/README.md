# Curso 2: Mejora de Redes Neuronales Profundas

En este segundo curso de la especialización, el enfoque es aprender a optimizar el rendimiento de las redes neuronales. Ya sabemos cómo construir una red profunda; ahora aprenderemos a hacer que entrene más rápido y lograr que generalice mejor.

## Temas principales

- **Ajuste de hiperparámetros (Hyperparameter Tuning):** Técnicas sistemáticas para encontrar la mejor combinación de valores (learning rate, cantidad de capas, etc).
- **Regularización (Regularization):** Métodos como L2, Dropout y Data Augmentation para evitar el sobreajuste (overfitting).
- **Optimización (Optimization):** Algoritmos avanzados que superan al Descenso de Gradiente tradicional, como Momentum, RMSprop y Adam.
